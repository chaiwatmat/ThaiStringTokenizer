# ThaiStringTokenizer

## Run test

```sh
dotnet test
```

## Build

```sh
dotnet build
```

## Publish to nuget.org

```sh
dotnet nuget push ThaiStringTokenizer/bin/Debug/ThaiStringTokenizer.0.2.1.nupkg \
    -k $NUGET_TOKEN \
    -s https://api.nuget.org/v3/index.json
```
